# Корочки.есть

Веб-приложение на PHP для записи пользователей на онлайн-курсы дополнительного профессионального образования.

## Возможности

- регистрация пользователей с проверкой логина, пароля, ФИО, телефона и email;
- авторизация пользователей и администратора;
- создание заявки на курс с выбором даты и способа оплаты;
- просмотр пользователем своих заявок;
- отзыв доступен только после статуса "Обучение завершено";
- панель администратора со списком всех заявок, фильтрацией, пагинацией и сменой статуса;
- адаптивный интерфейс в мягких бордово-красных цветах;
- полукруглая шапка с эффектом liquid-glass;
- слайдер из 4 изображений с автопереключением каждые 3 секунды.

## Технологии

- PHP 8+
- MySQL / MariaDB
- mysqli
- HTML, CSS, JavaScript

## Установка

1. Скопируйте проект в папку локального сервера.
2. Создайте базу данных, импортировав SQL-код из файла `database.txt`.
3. Проверьте настройки подключения в файле `config.php`:

```php
const DB_HOST = 'localhost';
const DB_USER = 'root';
const DB_PASSWORD = '';
const DB_NAME = 'korki_portal';
```

4. Откройте сайт в браузере через локальный домен или localhost.

## Доступ администратора

- логин: `Admin`
- пароль: `KorokNET`

У администратора в верхнем меню доступны только кнопки `Заявки` и `Выйти`.

## Доступ пользователя

После регистрации пользователь попадает в личный кабинет. В верхнем меню доступны только кнопки `Заявки` и `Подать заявку`.

## Структура проекта

- `index.php` - авторизация;
- `register.php` - регистрация;
- `requests.php` - заявки пользователя и отзывы;
- `create_request.php` - создание заявки;
- `admin/requests.php` - панель администратора;
- `includes/` - подключение к БД, авторизация, общие шаблоны;
- `assets/` - стили, скрипты и изображения;
- `database.txt` - SQL-код базы данных.
